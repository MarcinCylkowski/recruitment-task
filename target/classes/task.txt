1. Strona serwisu: https://www.worldweatheronline.com/developer/
2. Przeprowadzić rejestrację, darmowe konto - 60 dni
3. Przeczytać dokumentację i zapoznać się z możliwościami api.
4. Jest możliwych kilka opcji integracji, proszę wybrać opcję REST API
5. Program powinien wyświetlać pogodę na dany dzień dla Starogardu Gdańskiego w formacie:
Dzień : 2020-05-02
Maksymalna Temperatura : 9°C
Minimalna Temperatura : 8°C
Średnia Temperatura : 9°C

6. Testy jednostkowe będą na plus
7. Proszę o udostępnienie kodu za pomocą GitHub lub innego serwisu GIT.
